## Introduction

Have you ever had a fleeting encounter with another Rutgers stranger that left you with a sense of wonder and curiosity? We believe that these chance meetings are what make college life extraordinary. **Missed Connections Map** is a platform to relive those special moments and turn them into lasting connections ðŸ˜Š. [It's based on the popular instagram page](https://www.instagram.com/rutgers.missedconnections/).


## Features

- **Map-Based Connections**: Find and connect with students based on their location on the Rutgers campus. 

- **Create Missed Connections**: Share your own missed connections by dropping a pin on the map, describing the encounter, and adding relevant details.

## Technology Stack

Missed Connections Map is a mobile app built using React Native. The backend of the application was initially built with Rust but has since been transitioned to Laravel.

- **Frontend**: React Native
- **Backend**: Laravel (originally Rust)
- **Database**: Postgress
- **Maps**: Google Maps API
